Installation Instructions

To clone the repository and run the React frontend app, follow these steps:

  1. Open your terminal and navigate to the directory where you want to clone the repo.
  2. Run git clone https://github.com/Adam-Abinsha-vahab-Baker/React-Frontend.git
  3. Navigate into the cloned repository with cd your-repo-name
  4. Install the dependencies with npm install
  5. Start the React app with npm run dev

Database Instructions:

  • The connection string for the MongoDB database can be found in the .env file (which I am not going to gitignore).
  • If you are using the default database, you must share your machine IP with me, so I can allow access on the MongoDB website.
  • If you are going to create a database yourself, make sure to replace it with your MongoDB connection string.
  • The database name should be trip_management.
  • In the database, there should be two collections:
    • customers
    • trips
  • I highly recommend the first option (sharing your IP) to avoid complications.
